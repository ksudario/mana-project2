# Mana
An app to help parents or caregivers of kids with special needs to find resources that they need.

## Getting Started
[Click Here](https://mana-autism-resource.herokuapp.com) to get started

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Fonts
- MongoDB
- OAuth
- Heroku
